 # E-Commerce App

E-Commerce application with MVVM & Clean Arch - Coroutines - Retrofit - Firebase - Room DB.


## Project Features

- MVVM with Clean Architecture
- Coroutines
- Navigation Components
- Retrofit
- Glide
- Firebase Auth & Firestore
- Room Database
